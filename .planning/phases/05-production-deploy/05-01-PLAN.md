## Tasks

<task type="auto">
  <name>Increase Nginx Upload Limit</name>
  <files>nginx/default.conf</files>
  <action>
    1. Open `nginx/default.conf`.
    2. Change `client_max_body_size 10M;` to `client_max_body_size 50M;`.

  </action>

<task type="auto">
  <name>Add Resource Limits to Docker Compose</name>
  <files>docker-compose.yml</files>
  <action>
    1. Open `docker-compose.yml`.
    2. Under the `app` service, add a `deploy` section to limit resources to prevent memory leaks from crashing the host:
       ```yaml
       deploy:
         resources:
           limits:
             cpus: '1.0'
             memory: 1G
       ```
    3. Under the `db` service, add a similar section:
       ```yaml
       deploy:
         resources:
           limits:
             cpus: '1.0'
             memory: 1G
       ```
  </action>
  <verify>cat docker-compose.yml</verify>
  <done>Docker compose configuration includes explicit resource limits for `app` and `db` services.</done>
</task>
